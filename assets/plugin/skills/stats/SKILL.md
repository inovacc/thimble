# Stats — Knowledge Base Statistics

Call the `ctx_stats` MCP tool to display statistics about the thimble knowledge base.

This shows:
- Total indexed documents and size
- Per-project document counts
- Session event counts
- Database health metrics

Display the results in a clear, readable format.
